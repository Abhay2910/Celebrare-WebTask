Celebrare Web Task - Submission README

Files included:
- index.html
- css/styles.css
- js/script.js
- assets/logo.svg
- assets/hero.svg
- assets/img1.svg
- assets/img2.svg
- assets/img3.svg
- README_submission.txt (this file)

How to run locally:
1. Unzip the package.
2. Open index.html in your browser (double-click) OR run a simple static server:
   python -m http.server 8000
   and visit http://localhost:8000

How to create the submission ZIP (examples):
- macOS / Linux:
  zip -r celebrare-webtask.zip index.html css js assets README_submission.txt
- Windows PowerShell:
  Compress-Archive -Path index.html,css,js,assets,README_submission.txt -DestinationPath celebrare-webtask.zip

What to upload to the Google Form:
- The ZIP file (celebrare-webtask.zip) containing everything above.
- In the form fields include your name, email, LinkedIn URL, GitHub repo (optional), and a short description.

Good luck!
